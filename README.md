# Home Admin - Property Management System

Modern property management dashboard built with Next.js, React, Chakra UI, and PostgreSQL.

## Features

- Property management with simplified forms
- Utility expense tracking with provider, status, and invoice details
- Authentication with multi-user data isolation
- Dark themed UI optimized for desktop and mobile

## Available Scripts

- `npm run dev` - Start development server
- `npm run build` - Build for production
- `npm run start` - Start production server
- Apply schema manually: `psql -d your_database -f src/database/schema.sql`

Built with [Next.js](https://nextjs.org) and [Chakra UI](https://chakra-ui.com)
